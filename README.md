# Dashboard Power BI — Ligue 1 2025/2026

Dashboard interactif sur les statistiques de la Ligue 1 créé avec Power BI.

## Lien live
[Voir le dashboard en ligne](https://app.powerbi.com/groups/me/reports/c3533daf-1c93-441c-b75f-c4126901a177/cf4a71fed9add903a2db?experience=power-bi)

## Pages
- **Meilleur Buteur & Passeur** — Top 10 buteurs + Scatter buts vs passes
- **Stats par Club** — Classement clubs + Cartons jaunes/rouges  
- **Efficacite Offensive** — Scatter tirs vs précision

## Outils utilisés
- Power BI Desktop
- Power Query
- FBref (données réelles 2025/2026)

## Visualisations
- Bar chart Top 10 buteurs
- Scatter plot buts vs passes décisives
- Tableau classement des clubs
- Bar chart cartons jaunes/rouges
- Scatter tirs vs précision offensive
